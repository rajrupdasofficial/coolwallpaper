## About

Coolwallpaper  with Material design for setting cool wallpaper in your android phone. Written with kotlin, jetpack
compose,
other modern android libraries. Developing is still in progress and many features are not available now.

## Features

- Material  theme with adaptive color support
- Explore 40+ Wallpapers on random topic
- Set wallpapers on screen or lock screen
- view wallpaper at zoom
- Download wallpaper on external storage
